团购 Github 手办
===

Hi, all~ 上个周末参加了深圳动漫展，很遗憾忘了买一个手办回来。然后突然想到了 Github 官方的手办，一直想买的，但是总是忘记。于是下定决心，买一个回来放在桌前，改善一下生活质量，哈哈哈。

是这样的，团购价格不会打折，只是为了省一些邮费。官方的邮费略坑，直邮到国内单个邮费就要20刀，但是一次买多个的话邮费就会便宜些。试了一下，2个邮费大概是30刀，比直接买一个划得来一些。量更多，大家均摊的邮费也就越少。所以也就有了这一次团购。

如果有人感兴趣的话，欢迎一块去海淘一个回来，看了一下，这货真心不错~就是有些小贵，但是绝壁不是淘宝之流能比的。

流程是这样的，把你的收件地址和预付现金通过支付宝或者微信先转给我(信得过好说，信不过也没办法了...)，然后统一邮寄到我这里，最后我再通过快递寄给你（或者你有更好的办法？）。价格大概是（20刀商品原价 + 国际运费（试人数而定，单个是20刀，人数越多均摊下来就越少）+ 国内运费）。有意者欢迎通过邮件（地址你懂的）或者微信(longkai_simplicity)联系~

结束时间暂定于本周五（7月24号）晚上12点。
